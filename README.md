# Простой калькулятор на Python

Проект представляет собой простой калькулятор с базовыми арифметическими операциями и полным покрытием тестами.

## Функциональность

- ➕ **Сложение** (`add(a, b)`)
- ➖ **Вычитание** (`subtract(a, b)`)  
- ✖️ **Умножение** (`multiply(a, b)`)
- ➗ **Деление** (`divide(a, b)`) с проверкой деления на ноль
